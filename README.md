Код был написан в среде Pycharm Proffesional edition, всвязи с чем не было необходимости отдельно настраивать  venv  и создавать Django проект. 
Однако для людей, не имеющих такой возможности вот ссылка: https://metanit.com/python/django/1.2.php .
Там есть все нужные инструкции.
Наше приложение состоит из основного так называемого Z_sait(Zamechatelnii_sait), а так-же отдельного модуля NASHE_PRILOZHENIE(потому что для студсовета)
Файл зависимостей был создан недавно, их все можно посмотреть и установить.
Для запуска нашего приложения нам понадобится команда:
python3 manage.py runserver
После ввода этой команды нас забросит на начальную страничку нашего браузера. Которая никакого функционала на себе не несет. 
Для начала работы нам нужно перейти на страничку '/upload' для начала работы с приложением. Там мы загружаем картинку любого формата и размера(но хотя бы 512 на 512).
После загрузки изображение для его дальнейшей обработки мы переходим по адресу '/image-info' где выбираем координаты центра обрезки(от этой точки впоследствие будут отступы в 256 пикселей).
Важно! Координата y задается СВЕРХУ-ВНИЗ то есть: 0 - высшая точка изображения. С Х все вроде нормально.
Результат обрезки можно посмотреть по адресу '/kultya'
И в конце по адресу '/final' мы даем нашему стикеру эмодзи(есть вариант по умолчанию, однако можно взять и с тг) и одобряем отправку стикера.
После всех этих манипуляций стикер должен появиться в наборе https://t.me/addstickers/GOIDAZVONCHEK_336_by_Goidazvonchikbot
